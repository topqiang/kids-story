# 绘本：《宝宝初入幼儿园》

## 项目说明

这是第一集《宝宝初入幼儿园》的绘本版本，独立于后续动画视频工程。

- 绘本页数：10 页
- 图片目录：`images/`
- 正文文件：`storybook.md`
- 页面数据：`pages.json`
- 适合阅读对象：3-6 岁孩子和家长亲子共读

## 制作原则

- 每一页图片和剧情文案一一对应。
- 文案采用亲子绘本语气，不直接照搬视频对白。
- 图片不直接嵌入字幕，便于后续单独排版、配音或转动画。
- 后续动画视频请另建独立目录，不在本绘本目录内混放视频工程素材。

## 角色一致性

- 小北斗、婷婷妈妈、北斗爸爸均使用 `assets/characters/` 中的定稿角色形象。
- 婷婷妈妈统一使用白 T、牛仔裤、白色休闲鞋的年轻活力版。
- 若后续重生成页面图片，必须继续参考定稿角色图。
