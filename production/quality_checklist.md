# 质量检查清单

## 内容安全

- [ ] 没有危险模仿动作。
- [ ] 没有打骂、羞辱、恐吓式教育。
- [ ] 没有医疗、药品或疾病判断建议。
- [ ] 没有儿童隐私或不适合儿童观看的内容。
- [ ] 没有把撒谎、破坏规则、浪费食物包装成胜利。

## 剧情节奏

- [ ] 前 3 秒能看懂冲突。
- [ ] 45-90 秒内讲完一个完整故事。
- [ ] 至少有 2 个明确笑点。
- [ ] 结尾有反转、金句或温情落点。
- [ ] 对白短、清楚、适合配音。

## 视觉一致性

- [ ] 小北斗的发型、服装、脸型和颜色一致。
- [ ] 婷婷妈妈和北斗爸爸的造型一致。
- [ ] 场景符合家庭生活逻辑。
- [ ] 字幕没有遮挡人物关键表情。
- [ ] 手机端观看时表情清晰。

## 声音质量

- [ ] 对白清楚，音量稳定。
- [ ] 背景音乐不盖住对白。
- [ ] 音效数量适中，节奏点明确。
- [ ] 孩子声线自然，不尖锐。
- [ ] 家长语气真实但不凶。

## 发布检查

- [ ] 视频比例为 9:16。
- [ ] 封面包含角色和冲突文字。
- [ ] 标题口语化，能体现共鸣点。
- [ ] 文案没有夸大或误导。
- [ ] 已记录发布时间和版本号。
